1.2 Network Components

Host Roles

All device connected to a network and participate directly in network communication are classified as host. Host are devices that are assigned an Internet Protocole (IP) adress. the IP address identify the host and the network it's connected to.

Peer-to-Peer

A network configuration where the two end devices are directely connected together and have to act as Host and Servers.

The advantages of peer-to-peer networking:

  • Easy to set up
  • Less complex
  • Lower cost because network devices and dedicated servers may not be required
  • Can be used for simple tasks such as transferring files and sharing printers

The disadvantages of peer-to-peer networking:

  • No centralized administration
  • Not as secure
  • Not scalable
  • All devices may act as both clients and servers which can slow their performance

1.8 Network Security

Security Threats

Network security must consider environnement, tools and requirements of network while allowing quality of services expected from it.

There is external threats and internal threats. External comes from the internet while internal come from devices in the network.

External threats can include:

  • Viruses, worms, and Trojan horses - malicious software or code running on a user device
  • Spyware and adware - software which are installed on a device to secretly collects information about the user.
  • Zero-day attacks - Also called zero-hour attacks, occur on the first day that a vulnerability becomes known
  • Threat actor attacks - A malicious person attacks user devices or network resources
  • Denial of service attacks - These attacks slow or crash applications and processes on a network device
  • Data interception and theft - This attack captures private information from an organizationâ€™s network
  • Identity theft - This attack steals the login credentials of a user in order to access private data

Internal threats need to not be underestimated, since the BYOD strat is becoming popular.

Security Solutions

One solution is not enough to protect from all threats. Multiples ones should be used.

Basic solutions for homes and small office networks:

  • Antivirus and antispyware - Applications that help to protect end devices from becoming infected with malicious software.
  • Firewall filtering - Blocks unauthorized access into and out of the network. Could be a host-based firewall system that prevents unauthorized access to the end device, or a basic filtering service on the home router to prevent unauthorized access to the network.

Security solutions for corporations include the above and the following:

  • Dedicated firewall systems - Provide more advanced firewall capabilities that can filter large amounts of traffic.
  • Access control lists (ACL) - These further filter access and traffic forwarding based on IP addresses and applications.
  • Intrusion prevention systems (IPS) - These identify fast-spreading threats, such as zero-day or zero-hour attacks.
  • Virtual private networks (VPN) - These provide secure access into an organization for remote workers.

The study of network security threats and mitigation techniques starts with a clear understanding of the network structure.
