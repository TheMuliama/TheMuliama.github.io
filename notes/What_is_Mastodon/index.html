What is Mastodon

https://brookmiles.ca/2024/09/01/you-cant-join-mastodon/

Mastodon is a software project than let you create a server on the fediverse. It's not a social network, but rather a tool to create one. Each server you create with Mastodon can communicate with other server on the fediverse.

Server on the fediverse can communicate even if they don't have the same software. Mastodon server, PeerTube servers and Pixelfed can communicate together for example. You can subscribe to any account on the fediverse with only an account on a Mastodon server. https://blog.joinmastodon.org/2018/08/mastodon-quick-start-guide/
